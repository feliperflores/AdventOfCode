# AdventOfCode

This repository contains my solutions for [Advent of Code](https://adventofcode.com/) challenges, implemented in JavaScript.

## Running Solutions

### AdventOfCode2023

```bash
cd AdventOfCode2023
npm install

# Run tests
npm test

# Run specific day
npm run main:day1:first
```

### AdventOfCode2025

```bash
cd AdventOfCode2025
node day1/solve.js   # Runs Part 1
node day1/solve2.js  # Runs Part 2
```
